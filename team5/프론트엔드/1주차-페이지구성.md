# 페이지 구성

- 사용자
    1. 회원가입/로그인페이지
        1. 관리자/사용자로 나뉨
    2. 마이페이지
        1. 예약내역확인페이지
    3. 전체 물품 페이지 (메인화면)
        1. 메인화면에서 카테고리/토글로 나누는 형식
        2. 물품상세페이지
    4. 알림 - 모달
- 관리자
    1. 관리자 권한 부여 페이지
        1. 관리자 권한을 부여하게되면, 로그인페이지에서 관리자로 로그인이 가능하게됌.
    2. 물품관리페이지
        1. 물품상태관리 
            1. 사용자가 반납할때 찍은 물품사진을 확인하고 물품상태를 기록함
            
            → 상태가 별로면 제재/벌점
            
        2. 물품추가
        3. 물품삭제
        4. 재고관리
