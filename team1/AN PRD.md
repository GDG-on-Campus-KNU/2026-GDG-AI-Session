
# AN

모바일 특화 기능, 기기 대응 전략 수립

- 실시간 로컬 푸시 알림
    - 반납 시간 및 연체 알림
    - 예약한 대여 물품 반납 시 알림
- 대여 기록 및 이미지 캐싱
    - 네트워크가 없는 상황에서도 대여 정보를 확인
    - 불필요한 네트워크 사용 감소
- 생체 인증 로그인
    - 최초 로그인 후에는 지문/얼굴 인식을 통해 빠른 로그인
- QR코드 스캔(카메라 연동)
    - 물품마다 QR코드를 부착하여 스캔할 경우 관리자가 대여/반납 처리를 원클릭으로 진행할 수 있게 하기
- 다양한 화면 비율 및 해상도 대응
    - 기기마다 다른 화면 크기에 비율을 맞춰 UI 구성하기
- 직관적인 관리자용 화면
    - 재고 관리 화면 사용성
- 사용자 친화적인 UX Writing
    - 남은 대여 기간을 날짜로 표시하지 않고 몇 일 남았는지 등으로 표시
- 초기 사용자 가이드
    - 회원가입 후 로그인 유도
    - 사용 시 대여 한도/페널티 안내
- 다크모드 지원하기
    - 일관된 사용자 경험 제공
