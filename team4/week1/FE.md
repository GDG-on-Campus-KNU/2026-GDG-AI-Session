# Week 1 — 로그인/가입 & 물품 조회

## F-AU-01-FE — 로그인/가입 화면
- 이메일 입력 폼 및 정규식 검사 (`@knu.ac.kr` 도메인 확인) UI 구현
- 인증 메일 발송 요청 및 인증 코드 입력 화면 구현
- 로그인 성공 시 JWT / 세션 토큰 클라이언트 스토리지 저장

## F-DI-01-FE — 전체 목록 리스트
- 대여 가능 물품 리스트(그리드 / 리스트 뷰) UI 렌더링
- 무한 스크롤 또는 페이지네이션 UI 적용

## F-DI-02-FE — 카테고리 필터 UI
- [디지털/IT, 행사/레저, 도서, 기타] 탭 또는 드롭다운 UI 제공
- 필터 선택 시 백엔드 API 호출 및 화면 즉각 갱신

## F-DI-03-FE — 물품 상세 화면
- 물품 사진, 규격, 대여 가능 수량, 최대 기간 등의 텍스트/이미지 렌더링
- 수량이 0일 경우 예약 버튼 비활성화(Disabled) 처리
